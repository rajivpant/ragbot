## Design Principles

1. **RAG First**: RAG is enabled by default, not hidden
2. **Minimal Chrome**: Maximum space for chat content
3. **Progressive Disclosure**: Advanced settings in modal
4. **Status Visibility**: Index state always visible
5. **Mobile Friendly**: Top bar collapses gracefully
