# Compiler Usage

> CLI commands and usage examples for the AI Knowledge Compiler.

## Current Architecture (as of March 2026)

The compiler has been simplified to instructions-only. Knowledge concatenation moved to CI/CD.

| Operation | Command |
|-----------|---------|
| Compile instructions | `ragbot compile --project {name}` |
| Compile without LLM | `ragbot compile --project {name} --no-llm` |
| Index for RAG | `ragbot index --workspace {name}` |

Knowledge concatenation (`all-knowledge.md`) is handled automatically by GitHub Actions on every push to `source/`.

## Output Structure

```
ai-knowledge-{name}/
├── compiled/
│   └── {project}/
│       └── instructions/
│           ├── claude.md
│           ├── chatgpt.md
│           └── gemini.md
└── all-knowledge.md          # Auto-generated by CI/CD
```

## Using Compiled Output

### Claude Projects

1. Copy content from `compiled/{project}/instructions/claude.md`
2. Paste into Claude project custom instructions
3. GitHub sync the repo (Claude reads all-knowledge.md)

### ChatGPT

1. Copy from `compiled/{project}/instructions/chatgpt.md`
2. Upload `all-knowledge.md` from repo root

### Gemini Gems

1. Copy from `compiled/{project}/instructions/gemini.md`
2. Upload `all-knowledge.md` from repo root

### RAG (Qdrant)

```bash
ragbot index --workspace {name}
```

Reads source files directly, chunks, embeds, and indexes in Qdrant.

## Configuration

### compile-config.yaml

Each repo has a `compile-config.yaml` for compilation settings:

```yaml
project:
  name: example
  description: Example knowledge base
  output_dir: ./compiled

sources:
  local:
    path: ./source
    include:
      - "**/*"
    exclude:
      - "**/.DS_Store"
      - "**/README.md"

compilation:
  default_token_budget: 100000
  default_compiler:
    engine: anthropic
    model_category: flagship
  targets:
    - name: claude-projects
      platform: anthropic
      model_category: flagship
      output: compiled/claude-projects/
    - name: chatgpt-projects
      platform: openai
      model_category: flagship
      output: compiled/chatgpt-projects/
```

## Convention-Based Discovery

The compiler automatically discovers repositories by convention:

1. Looks for `ai-knowledge-*` directories
2. No explicit configuration needed
3. Default path: `$RAGBOT_BASE_PATH` or `~/ai-knowledge/`
