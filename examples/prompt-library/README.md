## How to Use These Prompts

### Method 1: Direct Usage

Copy any prompt file and use it directly in your conversation with Ragbot:

```bash
# Copy the content and paste into Ragbot
cat examples/prompt-library/prompt-engineering/tree-of-thought.md
```

### Method 2: Add to Custom Instructions

Incorporate these into your custom-instructions for consistent behavior:

```bash
# Append to your custom instructions
cat examples/prompt-library/ai-configuration/code-generation.md >> custom-instructions/my-instructions.md
```

### Method 3: Use as Templates

Customize these prompts for your specific needs:

1. Copy the prompt file to your workspace
2. Modify it for your use case
3. Save it in your curated-datasets or custom-instructions
